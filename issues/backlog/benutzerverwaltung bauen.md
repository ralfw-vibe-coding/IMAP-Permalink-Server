Wir brauchen noch ein paar Dinge für eine Benutzerverwaltung und zur Komplettierung der Authentifizierung:

- Bei Login muss man melden können, dass man das Passwort vergessen hat. Dann soll man über Angabe seiner Email-Adresse eine Rücksetz-Email bekommen.
- Aus der Rücksetz-Email soll man zu einem Formular kommen, wo man das neue Passwort eingeben kann.
- Benutzer sollen ihren Benutzernamen und ihr Passwort ändern können. Dafür müssen sie aus der Homepage zu einem Dialog kommen, der das zulässt.
- Nach Signup soll eine Confirmation Email kommen. Erst, wenn man bestätigt hat, kann man Emails in der INBOX sehen. Vorher darf man aber schon IMAP Server erfassen.
