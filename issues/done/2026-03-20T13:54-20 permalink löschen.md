In der Übersicht der Permalinks möchte ich einen Link löschen können.
Dafür sollte auch ein kleiner roter Button auf dem Link stehen neben den anderen.
Darauf ein kleines Icon (trash can).

---
Permalink-Liste um einen kleinen roten Trash-Button ergänzt. Dazu einen echten DELETE-Endpunkt im Server eingebaut und die UI so verdrahtet, dass der Eintrag nach dem Löschen sofort aus der Liste verschwindet. Während des Löschens zeigt der Button einen kleinen Spinner.
